# Intelligent Driver Risk Assessment and Adaptive Cockpit Safety Controller

Automotive-inspired embedded safety controller developed using Raspberry Pi Pico (RP2040) and Embedded C.

## Features

* Layered embedded software architecture
* Hardware Abstraction Layer (HAL)
* ADC-based sensor acquisition
* Risk assessment engine
* Finite State Machine (FSM)
* Interrupt-driven emergency handling
* RP2040 hardware watchdog supervision
* UART telemetry streaming
* Real-time Python cockpit dashboard
* Automotive-style circular gauges

## Hardware Used

* Raspberry Pi Pico (RP2040)
* Potentiometers
* LEDs
* Push Button
* Breadboard + Resistors

## Software Stack

* Embedded C
* Raspberry Pi Pico SDK
* Python
* PySerial
* CustomTkinter

## Dashboard

Displays:

* Vehicle speed
* Driver fatigue level
* Risk score
* System state
* Emergency status

## Embedded Concepts Demonstrated

* GPIO
* ADC
* UART
* Interrupts
* Watchdog Timers
* Finite State Machines
* Layered Architecture
* Embedded C
